A tag BODY, desta página HTML, tem o seguinte conteúdo:
		<body onclick="clickBody();" onkeypress="keyPress();">
	
		
		As seguintes funções estão definidas:
		
		function keyPress(){
			alert('Você pressionou uma tecla.');
			}		
		function clickBody(){
			alert('Você clicou na página do navegador.');
			}		
		
		Agora dê um clique do mouse (botão normal) em qualquer lugar desta página ou pressione alguma tecla do teclado.
